# 협의사항 관리 시스템 — 사용자 매뉴얼

> AI 기반 심사품질 협의사항 관리 시스템 with 포지AI & 반골이AI


## 1. 시작하기

### 로그인
1. 접속
2. 아이디와 비밀번호 입력
3. 계정이 없으면 "계정 생성" 버튼으로 가입 (관리자 승인 필요)

### 화면 구성
- **상단 네비게이션**: 목록, 새 협의사항, 관계도, AI 업무편지함, 통계, 다운로드
- **🌙 버튼**: 라이트/다크 모드 전환 (눈의 피로 감소)
- **우측 상단**: 프로필, 비밀번호 변경, 로그아웃

---

## 2. 대시보드

메인 화면입니다. 모든 협의사항이 최신순으로 표시됩니다.

### 검색
- **일반 검색**: 제목, 내용, 키워드로 검색
- **AI 검색**: 자연어로 검색 (예: "청구항 해석 관련 부적합 사례")
- **필터**: 진단코드, 등급(부적합/주의/하), 관점(절차/명세서해석/검색/고객지향성)

### 목록 보기
| 항목 | 설명 |
|------|------|
| 🟢 **키워드 배지** | 클릭하면 해당 키워드로 검색 |
| 💬 **댓글** | "N개 댓글" 클릭 → 미리보기 |
| 👍👎 | 좋아요/싫어요 수 |
| 💡 **유사** 버튼 | AI가 유사한 협의사항을 찾아줍니다 |



## 3. 협의사항 작성

`새 협의사항 작성` 버튼 클릭

### AI 다듬기
- "AI가 멋있게 다듬어서 올리기" 체크 → AI가 맞춤법·문법·전문용어 교정
- 미리보기로 결과 확인 후 적용 가능

### 키워드 태그
- `AI 키워드 추천받기` 클릭 → AI가 내용 분석 후 7개 키워드 제안
- 원하는 키워드 클릭 → 선택됨 (녹색 배지)
- 마음에 안 들면 ✕ 클릭으로 제거
- 선택 안 하면 AI가 자동 생성

### 진단코드 & 관점 (선택)
- 진단코드 4자리 (예: 0531)
- 3번째 숫자로 등급 자동 결정: 1=부적합, 2=주의, 3=하
- 관점: 절차 / 명세서 해석 / 검색 / 고객지향성

### 파일 첨부
- 드래그 앤 드롭 또는 파일 선택
- 지원: TXT, MD, PDF, 이미지, 문서 (최대 10MB)
- README.md 파일은 GitHub 스타일로 렌더링

---

## 4. 협의사항 보기

게시글을 클릭하면 상세 페이지가 열립니다.

### 본문
- 제목, 내용, 진단코드, 등급, 관점, 작성자, 작성일
- 좋아요/싫어요 버튼
- 첨부파일 (마크다운 렌더링, 이미지 미리보기)

### 💚 포지AI 의견 (찬성)
- 게시글의 장점과 타당성을 보강하는 의견

### ❤️ 반골이AI 의견 (반대)
- 게시글의 논리적 허점과 추가 검토사항 지적

### 댓글
- 댓글 작성 및 삭제 (내 댓글만)
- AI 답글 버튼: AI가 댓글 내용을 분석하여 답글 작성

### 의사결정 투표
- 동의 / 이의 / 보류 버튼
- 실시간 투표 현황 표시

---

## 5. AI 유사 사례

### 대시보드에서
- 각 게시글 옆 `💡 유사` 버튼 클릭 → 팝업으로 유사 협의사항 표시
- 첫 분석 시 AI가 검색하여 DB에 저장 (약간의 대기)
- 두 번째부터는 **캐시된 결과 즉시 표시** (기다릴 필요 없음)
- `재분석` 링크로 최신 결과로 갱신 가능

### 게시글 상세에서
- 우측 사이드바 `분석` 버튼 → 유사 사례 검색
- 캐시된 결과가 있으면 페이지 로드 시 자동 표시

---

## 6. 키워드 관계도

`관계도` 탭에서 접근

- AI가 생성한 키워드 기준으로 협의사항 그룹핑
- 같은 키워드를 가진 게시글끼리 묶어서 표시
- `↔` 버튼으로 두 게시글 나란히 비교
- 키워드 검색으로 원하는 주제 찾기

---

## 7. AI 토론방

게시글 상세 페이지에서 `포지·반골이와 실시간 토론하기` 버튼

- 하나의 질문에 **포지와 반골이가 각자 입장에서 동시 답변**
- `Enter` 키로 전송, `Shift+Enter`로 줄바꿈

---

## 8. AI 업무편지함

`AI 업무편지함` 탭에서 접근

### 편지
- 포지나 반골이에게 개인 편지 쓰기
- AI가 편지 내용을 읽고 진심 어린 답장 작성
- HTML / Markdown / TXT 형식으로 보기 가능

### 업무
- 포지나 반골이에게 분석 업무 지시
- AI가 협의 데이터와 RAG 지식을 활용하여 보고서 작성
- 상태: 대기중 → 진행중 → 완료

---

## 9. 통계

`통계` 탭에서 접근

- 총 협의사항, 참여 인원, AI 의견 수
- 등급별 / 관점별 도넛 차트
- 월별 게시글 추이 바 차트
- 인기 키워드 TOP 10
- `AI 보고서 생성` 버튼 → AI가 통계 분석 보고서 작성 (다운로드 가능)

---

## 10. 다운로드

`다운로드` 메뉴에서 형식 선택

| 형식 | 용도 |
|------|------|
| **CSV** | 엑셀 호환, 데이터 분석 |
| **JSON** | 프로그램 연동, 백업 |
| **HTML** | 웹 브라우저에서 보기 |
| **Markdown** | 문서화, GitHub 업로드 |

게시글 상세에서도 개별 다운로드 가능 (JSON/HTML/MD).

---

## 11. 프로필 설정

우측 상단 `프로필` 메뉴

- **표시 이름**: 게시글과 댓글에 표시되는 이름 (언제든지 변경 가능)
- **아바타**: 제공된 이미지 중 선택

## 12. 팁 & 숏컷

| 팁 | 설명 |
|-----|------|
| **자연어 검색** | "청구항 해석 관련 부적합 사례"처럼 문장으로 검색 가능 |
| **키워드 클릭** | 게시글 아래 키워드 배지 클릭 → 같은 키워드 게시글 검색 |

| **드래그 앤 드롭** | 파일을 첨부 영역에 끌어놓기만 하면 업로드 |
| **PWA** | 모바일에서 "홈 화면에 추가"하면 앱처럼 사용 가능 |
